# commit_template
bannzai's commit template.

## Emoji
- 🐛  :bug: バグ修正
- 🤘  :metal: 機能改善
- 🐚  :shell: コマンドを実行した結果とか
- ✨  :sparkles: 部分的な機能追加
- 🎉  :tada: 盛大に祝うべき大きな機能追加
- 💉  :syringe: libraryの追加
- 📝  :memo: ドキュメント・READMEの整備
- ♻️   :recycle: リファクタリング
- 🚿  :shower: 不要な機能・使われなくなった機能の削除
- 💚  :green_heart: テストやCIの修正・改善
- 👕  :shirt: Lintエラーの修正やコードスタイルの修正
- 🚀  :rocket: パフォーマンス改善
- 🆙  :up: 依存パッケージなどのアップデート
- 🔒  :lock: 新機能の公開範囲の制限
- 👮  :cop: セキュリティ関連の改善
- 🎨  :art: デザイン修正
- 🚧  :construction: WIP 一時的なコミット
